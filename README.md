## 🖥️ Deploy on VPS / Local

**Step 1 — Clone the repo:**
```bash
git clone https://github.com/vishalpandeynkp1/VIPNOBITAMUSIC_REPO
cd VIPNOBITAMUSIC_REPO
```

**Step 2 — Setup:**
```bash
bash setup
```

**Step 3 — Fill your variables:**
```bash
nano .env
# Save: Ctrl+X → Y → Enter
```

**Step 4 — Install tmux & run:**
```bash
sudo apt install tmux && tmux
bash start
```

**Step 5 — Detach tmux session:**
```
Press Ctrl+B then D
```
